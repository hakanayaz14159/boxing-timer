# PWA Icons

Add these PNG icons to this directory. All must be square.

| Filename      | Size    | Purpose                          |
|---------------|---------|----------------------------------|
| `icon-192.png`| 192×192 | Android (manifest)               |
| `icon-512.png`| 512×512 | Android (manifest, maskable)    |
| `icon-180.png`| 180×180 | iOS (apple-touch-icon)           |
